# Food Picker App

A modern React + TypeScript + Tailwind CSS app that lets users pick a cuisine and receive a random dish suggestion.

## Live Demo

https://food-picker-app-weld.vercel.app

## Features

- Random dish generator
- React state management
- Keyboard Enter support
- Error handling
- Responsive modern UI
- Tailwind CSS styling

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
